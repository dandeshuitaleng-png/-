# NearBite 3.0 版本说明

更新时间：2026-07-01

## 1. 版本定位

`3.0-参赛包/iOS原型` 是在 2.0 基础上继续迭代的比赛展示版本。

本次调整的目标，不是推翻原有结构，而是在保留 2.0 内容的同时，把已经验证过的交互优化单独沉淀成 3.0，避免版本混用。

## 2. 与 2.0 的关系

- `2.0-参赛包/iOS原型` 保留原始 2.0 原型版本。
- `3.0-参赛包/iOS原型` 保存当前最新迭代版本。
- 两个版本并存，方便对比、回看和后续继续迭代。

## 3. 3.0 当前重点优化

- 登录注册首段流程更顺滑，补充了游客体验和跳过资料入口。
- 兴趣偏好页不再只是展示，而是会同步影响默认推荐。
- 推荐结果页新增反馈状态展示，让“收藏 / 已吃 / 近期不推荐”形成闭环。
- “我的”页面强化了偏好编辑、反馈汇总和历史回流。

## 4. 当前建议

后续如果继续开发，建议统一以 `3.0-参赛包/iOS原型` 作为最新演示版本推进；`2.0` 作为留档和对照版本保留。
